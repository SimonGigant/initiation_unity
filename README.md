Mon projet consistait en 2 aspects ;
- Créer une petite library de gamefeel (screenshake qui suit une animation curve,
	effet vertigo qui fonctionne sur une caméra fixe uniquement (à améliorer),
	effet de glitch pendant le screenshake (shader trouvé sur le net)
- Avoir un rendu esthétique qui mélange 2D et 3D (tourner le sprite 2D en fonction
	de l'orientation de la caméra, post process, déplacement 3D, caméra avec
	cinemachine)

Je n'avais pas beaucoup d'expérience dans Unity, mais cette semaine m'a permis d'être
plus à l'aise dans ce moteur, notamment avec la 3D. Elle m'a permis aussi d'être
opérationnel pour la Ludum Dare et l'Enjam que j'ai fait par la suite.
